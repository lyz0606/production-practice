# Стажировка на производстве – Программа задания 6

Эта программа используется для расчета статуса выполнения сменных заданий и была разработана во время стажировки в Самарской нефтегазовой компании (АО «Самара нефте»).

## Функции

- Ввод названия продукта, запланированного объема производства, фактического объема производства

- Расчет процента выполнения и определение, выполнен ли план

- Добавление результатов в файл `shift_report.txt`
